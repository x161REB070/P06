Secinājumi


Šajos darbos esmu novērojis, ka visvienkāršāk un ātrāk veidot dokumentus ir lietojot MS Word. Lai gan ērti un vienkārši,tas nebūt nenozīmē,ka nerodas problēmas to lietojot.
Rakstot gan tekstietilpīgus, gan pievienotiem attēliem bagātus dokumentus,pētījumus vai jebkāda tipa rakstus,ar laiku visi sastopas ar problēmām,kuras rodas MS Word. Piemēram, izmainot vienu rindu, vai kāda vārda izmēru, pārvietojot bildes ir manīts, ka viss dokuments pārvēršas haosā. Atkarībā no raksta lieluma, šī var būt sekundēs risināma problēma, vai pat stundām ilgas mokas.
Ja salīdzinām LaTeX ar MS Word, paaugstinās iemaņu un zināšanu līmenis,kuras nepieciešamas,lai darbotos ar LaTeX. Lai gan sarežģītāk veidot, raksti,it īpaši garas publikācijas vai citi raksti tomēr labāk veidotar LaTeX, jo var visu tekstu formatēt programmējot TeX un viena netīša atstarpe nesabojās visu darbu. Tajā var gan ievietot skaistas matemātiskas formulas,gan veidot grafikus, elektroniskās shēmas un daudz citu objektu,kurus izmantojot MS Word, būtu jāveido citās programmās.
Šīs priekšrocības nāk līdz arī veidojot HTML dokumentus,jo visus datus, attēlus un tekstu formatējam programmējot un varam tos novietot kur un kā vēlamies. Vēl liels plus HTML ir tā plašā izmantotība mājaslapu veidošanā, JavaScript un CSS izmantošanai,kas HTML bagātina ar daudz jaunām iespējām.

Īsumā:

MS Word - ātrs un vienāršs īsiem dokumentiem;
LaTeX - daudz labāks zinātnisko dokumentu veidošanai, sarežģītāks, bet atmaksājas ilglaicīgā lietošanā;
HTML - līdzīgi kā LaTeX, bet var vēl veidot mājaslapas.
